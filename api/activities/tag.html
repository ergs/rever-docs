Tag Activity (rever.activities.tag)¶

Activity for bumpimg version.

class rever.activities.tag.Tag(*, deps=frozenset({'changelog', 'version_bump'}))¶

Tags the current version and pushes the tag up to a remote repository.

This activity takes the following parameters:

$TAG_TEMPLATE:str, the template string to tag the version with, by default this is ‘$VERSION’
$TAG_PUSH:bool, flag for whether or not the current head and tag sould be pushed, default True.
$TAG_REMOTE:str or None, remote URL to push tags to. The default is None, which is only valid when push=False.
$TAG_TARGET:str or None, remote branch to push to once the tag has been made. The default is None, which uses the current branch.
